# Portfolio Showroom

Status: In progress. 

A 3D portfolio space built with Three.js. Walk around, explore different rooms, and inspect objects.

[Live Demo](https://showroom-prototype.pages.dev/)

![demo](./demo.webp)

## Controls

- WASD to move
- Mouse to look around
- I to inspect objects
- Escape for menu

## Tech

Three.js, GSAP, custom GLSL shaders, post-processing
